## Cron 任务配置

### 任务1：内容生成并发布（每天 9:00）

```bash
# 创建任务
cron add \
  --name "xiaohongshu-generate" \
  --expr "0 9 * * *" \
  --delivery-mode announce \
  --channel telegram \
  --sessionTarget isolated \
  --context-messages 3
```

**任务消息：**
```
执行小红书内容生成和发布任务：

1. 访问飞书多维表"热门关键词库"，读取状态为"可用"的3-5个关键词
2. 对每个关键词生成标题和文案
3. 保存到飞书多维表"小红书实验记录"
4. 通过小红书 MCP 发布笔记
5. 记录笔记链接

关键词：随机选择
风格：随机选择

注意：
- 避免重复关键词
- 不同时间发布（间隔2小时以上）
- 内容要有差异化
```

### 任务2：数据收集（每天 20:00）

```bash
# 创建任务
cron add \
  --name "xiaohongshu-collect" \
  --expr "0 20 * * *" \
  --delivery-mode announce \
  --channel telegram \
  --sessionTarget isolated
```

**任务消息：**
```
执行小红书数据收集任务：

1. 访问飞书多维表"小红书实验记录"
2. 筛选今天已发布的记录
3. 读取笔记数据（点赞、收藏、评论、转发）
4. 提取 Top3 评论
5. 更新数据字段
6. 计算互动率

注意：
- 如果笔记链接无效，标记为"数据异常"
- 所有数据必须记录来源
```

### 任务3：数据分析（每天 21:00）

```bash
# 创建任务
cron add \
  --name "xiaohongshu-analyze" \
  --expr "0 21 * * *" \
  --delivery-mode announce \
  --channel telegram \
  --sessionTarget isolated
```

**任务消息：**
```
执行小红书数据分析任务：

1. 读取今天所有有数据的记录
2. 分析评论情感
3. 计算数据指标
4. 生成优化建议

### 决策规则：

自动采纳（满足任一）：
- 互动率 > 10%
- 点赞数 > 100
- 收藏数 > 50
- 正面评论 > 80%

部分采纳（满足任一）：
- 互动率 5-10%
- 点赞数 50-100
- 收藏数 20-50
- 正面评论 60-80%

忽略不采纳（任一）：
- 互动率 < 5%
- 点赞数 < 50
- 收藏数 < 20
- 负面评论 > 40%

生成优化建议并保存到飞书多维表
```
