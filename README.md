# 小红书爆款文案生成器

## 产品定位

**目标用户：** 小红书运营者、带货博主、引流账号
**核心价值：** 3秒生成爆款文案，节省90%创作时间

## 功能清单

### MVP（第1版）
- [x] 关键词输入
- [x] AI生成标题（10条）
- [x] AI生成正文（3条）
- [x] 自动生成标签（5-10个）
- [x] 一键复制

### V2（增强版）
- [ ] 竞品爆款分析
- [ ] 图片建议
- [ ] 历史记录
- [ ] 批量生成

## 技术架构

**后端：**
- Node.js + Express
- Claude API（文案生成）
- 小红书MCP（数据抓取）

**前端：**
- 简洁Web界面
- 飞书小程序版本

**自动化：**
- OpenClaw cron 定时任务
- 飞书推送
