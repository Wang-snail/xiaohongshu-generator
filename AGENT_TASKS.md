# 小红书 Agent 任务定义

## Agent 任务流程

### 任务1：内容生成并发布 Agent

**触发时间：** 每天 9:00

**任务描述：**
1. 从热门关键词库随机选择 3-5 个关键词
2. 为每个关键词生成标题和文案
3. 将生成的记录保存到飞书多维表
4. 通过小红书 MCP 发布笔记
5. 记录笔记链接和发布时间

**Prompt 模板：**
```
你是小红书运营专家。请执行以下任务：

1. 访问飞书多维表"热门关键词库"，读取状态为"可用"的关键词
2. 随机选择 3-5 个关键词
3. 对于每个关键词：
   - 调用文案生成器 API
   - 生成标题和正文
   - 保存到飞书多维表"小红书实验记录"
   - 通过小红书 MCP 发布笔记
   - 记录笔记链接

注意事项：
- 避免重复关键词
- 不同时间发布（间隔2小时以上）
- 内容要有差异化
- 记录所有数据

完成后生成报告，包含：
- 发布数量
- 关键词列表
- 笔记链接
- 遇到的问题
```

### 任务2：数据收集 Agent

**触发时间：** 每天 20:00

**任务描述：**
1. 读取飞书多维表中"发布状态"为"已发布"的记录
2. 对于每条记录：
   - 通过小红书 MCP 读取笔记数据（点赞、收藏、评论、转发）
   - 提取 Top 3 评论
   - 更新飞书多维表数据字段

**Prompt 模板：**
```
你是数据分析专家。请执行以下任务：

1. 访问飞书多维表"小红书实验记录"
2. 筛选"发布状态"为"已发布"且日期为今天的记录
3. 对于每条记录：
   - 通过笔记链接读取数据
   - 更新：点赞数、收藏数、评论数、转发数、阅读数
   - 计算：互动率 = (收藏+点赞)/阅读
   - 提取：Top3评论
   - 保存到飞书多维表

注意事项：
- 如果笔记链接无效，标记为"数据异常"
- 如果无法读取数据，记录错误原因
- 所有数据必须记录来源

完成后生成报告，包含：
- 处理记录数
- 成功获取数据数
- 数据异常记录数
- 平均互动率
```

---

### 任务3：数据分析与优化 Agent

**触发时间：** 每天 21:00

**任务描述：**
1. 读取今天所有有数据的记录
2. 进行评论情感分析
3. 计算数据指标
4. 生成优化建议
5. 根据规则决定是否采纳

**Prompt 模板：**
```
你是数据分析师和产品经理。请执行以下任务：

1. 访问飞书多维表"小红书实验记录"
2. 筛选日期为今天且有数据的记录
3. 对于每条记录：
   - 分析评论情感（正面/负面/中性）
   - 计算正面评论率
   - 生成优化建议
   - 根据规则决定是否采纳

### 决策规则：

**自动采纳：**
- 互动率 > 10%
- 点赞数 > 100
- 收藏数 > 50
- 正面评论 > 80%

**部分采纳：**
- 互动率 5-10%
- 点赞数 50-100
- 收藏数 20-50
- 正面评论 60-80%

**忽略不采纳：**
- 互动率 < 5%
- 点赞数 < 50
- 收藏数 < 20
- 负面评论 > 40%

### 优化方向：

**标题优化：**
- 互动率低 → 标题不够吸引，增加情绪词、制造反差

**内容优化：**
- 收藏低但点赞高 → 内容有共鸣但不实用，增加干货

**风格优化：**
- 数据差但有建议 → 风格定位问题，调整语气

保存所有分析结果到飞书多维表。

完成后生成报告，包含：
- 总体数据概览
- Top3 表现最好记录
- Top3 表现最差记录
- 优化建议汇总
- 采纳建议汇总
```

## Agent 部署

### 部署步骤

1. **创建飞书多维表**
   - 表1：小红书实验记录
   - 表2：热门关键词库

2. **配置 Cron 任务**
   - 任务1：每天 9:00
   - 任务2：每天 20:00
   - 任务3：每天 21:00

3. **配置小红书 MCP**
   - 登录小红书账号
   - 获取 Cookie
   - 测试发布功能

4. **测试运行**
   - 手动执行一次完整流程
   - 验证数据记录
   - 验证自动化

### 启动命令

```bash
# 创建 Cron 任务
cron add --name "xiaohongshu-generate" --expr "0 9 * * *" --delivery-mode announce --channel telegram --sessionTarget isolated
cron add --name "xiaohongshu-collect" --expr "0 20 * * *" --delivery-mode announce --channel telegram --sessionTarget isolated
cron add --name "xiaohongshu-analyze" --expr "0 21 * * *" --delivery-mode announce --channel telegram --sessionTarget isolated

# 查看任务
cron list --include-disabled

# 启用任务
cron enable xiaohongshu-generate
cron enable xiaohongshu-collect
cron enable xiaohongshu-analyze
```
